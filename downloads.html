Mobster downloads

Mobster main software

It is strongly recommended to get Mobster version 2.5.0 or higher. The latest version can output VCF files directly; has support for almost all BAM files and can predict the target site duplication, variant allele fraction and somatically occurring insertions.


Older versions

  • Mobster version 0.1.6: Release date 13th of April 2013

Mobster utility software

  • MobsterToVCF 0.0.1: Utility to convert the Mobster prediction .txt files to .vcf. Not needed when using version 2.5.0 or higher.
